How To Use
01

Upload halaman ini ke hosting gratis

Gunakan GitHub Pages, Netlify, atau Vercel — semuanya gratis. Cukup upload file HTML ini, kamu dapat URL publik seperti https://yourname.github.io/starcore-bridge

02

Paste URL YouTube & klik Save

Masukkan URL video YouTube (maks 1 menit untuk best experience loading screen). Klik Save & Generate Link — link bridge akan terbuat otomatis dengan video ID tersimpan di URL.

03

Copy link ke Roblox Script

Copy link yang ter-generate, lalu paste ke variabel BRIDGE_URL di Roblox LocalScript. Script akan auto-load video dari bridge ini.

04

Ganti video kapan saja

Kamu bisa kembali ke halaman ini, masukkan URL baru, dan update link di script Roblox. Tidak perlu re-upload ke Roblox sama sekali.

Roblox Script — Variable to Update
-- Paste link ini di LocalScript kamu:
local BRIDGE_URL = "https://yourpage.github.io/starcore-bridge/?v=VIDEO_ID"
